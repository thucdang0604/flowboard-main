# 🚀 Flowboard v1.1.0 — Bản cập nhật lớn nhất tới giờ

Vừa release Phase 20 với hàng loạt tính năng mới và fix quan trọng. Anh em đang dùng Flowboard nhớ cập nhật để pickup những thay đổi này nhé!

## ✨ Tính năng mới

**👤 Hiển thị thông tin tài khoản tự động**
Không cần config tay nữa — extension tự động lấy email, tên, ảnh đại diện từ Google + nhận diện gói **PRO / ULTRA** của anh em. Hiển thị ngay ở góc dưới bên trái sidebar.

**🎨 Chọn model linh hoạt**
- Image: **Nano Banana Pro** (chất lượng cao) hoặc **Nano Banana 2** (nhanh, nhẹ)
- Video: **Veo 3.1 Fast** (mặc định) hoặc **Veo 3.1 Lite** (render nhanh hơn)
- Tất cả nằm trong dialog Settings (bấm icon ⚙)

**❤️ Sponsor button**
Bấm nút **♥ Sponsor** ở thanh toolbar → mở dialog premium với 4 tier: Coffee $5, Gold $25, Platinum $50, Diamond $100. Hỗ trợ cả Ko-fi (toàn cầu) và PayPal.

**🔔 Thông báo version mới**
Khi có release mới trên GitHub, AccountPanel sẽ hiện pill gradient pulse-animated → bấm vào xem changelog ngay.

## 🛡️ Fix quan trọng

- **Khắc phục bug mất aspect ratio**: trước đây nhiều image node bị mất tỉ lệ 9:16 khi auto-brief chạy → đã backfill 98 nodes cũ + sửa contract PATCH ở backend
- **Quick Regenerate** giờ giữ đúng aspect, variant count, multi-source upstream
- **Project sidebar** nhớ board đang chọn qua reload page
- **Captcha resilient**: không còn bị "No current window" khi Flow tab bị Chrome discard

## 🎯 QoL khác

- 6 vibe presets cho character builder (Clean Girl, Douyin, Old Money, Cold Girl, K-Pop, Casual)
- Multi-subject auto-prompt cho couple/group shots
- Video tile preview lấy ảnh nguồn upstream → composition giống image tile
- Prompt + Note nodes editable inline
- Toolbar wordmark viết hoa đúng: **Flowboard**

## 📦 Cách cập nhật

```bash
git pull
make update
# Sau đó reload extension ở chrome://extensions/
```
